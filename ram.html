ramminnet
RAM-minnet är datorns korttidsminne. När ett program startas laddas det in från hårddisken och sparas i ram minnet. Ibland kan den kallas också (Random Access Memory) för den engelska språket. Ett ram minne måste alltid vara strömsatt för att kunna fungera. Detta innebär att det töms så fort datorn stängs av och därför måste den ha ström för att kunna fungera. Datorn måste också ha tillräckligt mycket ram minne för att aktiva program och dokument ska kunna få plats i det. Det finns olika typer av minne som lagringsminne, arbetsminne, och ram minne. Därför är ramminne dyrare och snabbare än lagringsminne. Ram minnets jobb är att lagra data som processorn behöver för stunden eller inom en period i framtiden. Varje gång datorn startar hämtas operativsystemets viktigaste delar från hårddisken och läggs i RAM-minnet och på så sätt blir de lättåtkomliga. Ju större RAM-minne datorn har, desto mer information kan den lagra temporärt med snabb åtkomst. Det innebär att fler program kan vara igång samtidigt.
